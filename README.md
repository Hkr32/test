## Deployment
#### Create project:
```
composer create-project Igor-Art/test test-app
```
OR
```
composer install
cp .env.example .env
```
#### Set database settings in .env
#### Run installation script:
```
php install.php
```
#### Run php server:
```
php -S 127.0.0.1:8000 -t ./public
```
