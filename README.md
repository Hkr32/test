## Deployment
#### Create project:
```
composer create-project Igor-Art/test test-app
```
OR
```
composer install
cp config.example.php config.php
```
#### Set database settings in config.php  
#### Run installation script:
```
php install.php
```
#### Run php server:
```
php -S 127.0.0.1:8000 -t ./public
```
